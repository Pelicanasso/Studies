Parâmentro Action

Esse parâmetro entra na tag do form e define qual ação será tomando quando o botão submit for clicado, como por exemplo ser enviado para um documento PHP

Parâmetro Target

Esse parâmetro é colocado no form para definir aonde será aberta a nova janela após ser enviado o formulário

Value Description
_blank A resposta será exibida em uma nova guia ou janela
_self A resposta será monstrada na janela atual
_parent A resposta será exibida em um frame parente
_top A resposta será mostrada no corpo todo da janela
framename A resposta será mostranda em um iframe

Parâmetro GET

Esse parâmetro pode ser usado na tag form para especificar o método HTTP que será usado quando eviado a informação, tendo dois tipos "get" e "post". ainda não faço a menor ideia o que isso significa tenho que estudar o HTTP
