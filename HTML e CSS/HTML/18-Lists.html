Listas

podemos criar listas ordenadas e não ordenadas

para não ordenadas utilizamos o UL

  • não
  • sei
  • o
  • que
  • colocar

para listar ordenadas podemos utilizar o OL

  1. ainda
  2. não
  3. sei
  4. o
  5. que
  6. colocar

Lista de termos

bebe não mano é ruim

AAAAAA

Aplicando estilos as listas

Podemos por exemplo alterara o icone da lista

  • disc - bolinha preenchida
  • Circle - circulo sem preenchimento
  • square - quadrado fodase
  • none - desativa
